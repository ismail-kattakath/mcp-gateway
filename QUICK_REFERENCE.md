# MCP Gateway - Quick Reference

## Essential Commands

```bash
# Verify setup is complete
./scripts/verify-setup.sh

# Run all tests
./scripts/test.sh

# Start development server (with hot-reload)
./scripts/start.sh

# Start production server (with Docker)
./scripts/start-prod.sh

# Re-run initial setup (if needed)
./scripts/setup.sh
```

## Endpoints

| Endpoint | Purpose |
|----------|---------|
| `http://localhost:3000/health` | Health check |
| `http://localhost:3000/sse` | SSE connection for MCP clients |
| `http://localhost:3000/message` | MCP JSON-RPC messages |
| `http://localhost:3000/api/status` | Backend status |
| `http://localhost:3000/api/config` | Registry configuration |
| `http://localhost:3000/` | Web dashboard |

## Client Configuration

Add to your AI tool's MCP config:

```json
{
  "mcpServers": {
    "gateway": {
      "url": "http://localhost:3000/sse",
      "transport": "sse"
    }
  }
}
```

**Location:**
- Claude Code: `~/.claude/.mcp.json`
- Claude Desktop: `~/Library/Application Support/Claude/claude_desktop_config.json`

## Configuration Files

| File | Purpose | Location |
|------|---------|----------|
| `.env` | Secrets & environment variables | Project root |
| `registry.json` | Backend definitions | Project root |
| `~/.mcp/tokens.enc` | Encrypted OAuth tokens | Home directory |
| `~/.mcp/logs/gateway.log` | Server logs | Home directory |

## Enabled Backends

1. **obs** - OBS Studio Control
   - Requires: `OBS_WEBSOCKET_PASSWORD` in .env
   - Tools: `obs/start_recording`, `obs/stop_recording`, etc.

2. **kapture** - Screen Capture
   - No configuration required
   - Tools: `kapture/screenshot`, `kapture/record`, etc.

## Quick Troubleshooting

```bash
# Check Node.js version (need >= 18.0.0)
node --version

# Check for port conflicts
lsof -i :3000

# Validate registry
cd server && npm run validate

# View logs with debug level
LOG_LEVEL=debug ./scripts/start.sh

# Check backend status via API
curl http://localhost:3000/api/status
```

## Environment Variables

Key variables in `.env`:

```bash
# Gateway
GATEWAY_PORT=3000
GATEWAY_HOST=0.0.0.0
GATEWAY_API_KEY=<auto-generated>

# Security
TOKEN_ENCRYPTION_KEY=<auto-generated>
ENABLE_AUTH=false  # Set to true for production

# Backend Secrets
OBS_WEBSOCKET_PASSWORD=<set-your-password>

# OAuth (optional)
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
```

## Backend Management

Enable/disable backends in `registry.json`:

```json
{
  "backends": {
    "backend-name": {
      "enabled": true  // Change to false to disable
    }
  }
}
```
